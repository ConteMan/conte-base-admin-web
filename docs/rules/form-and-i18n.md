# 表单、详情与 i18n 规则

## 表单说明

- `label` 负责表达字段本身是什么。
- `help` 负责承载格式要求、默认行为、风险提示与副作用。
- 灰色辅助文字只用于较长且需要持续可见的提醒。
- `placeholder` 默认只用于示例输入，不作为正式字段说明。

## 详情展示

- 详情态不复用禁用表单。
- 禁止让 placeholder、校验态样式或伪交互残留在详情界面中。
- 优先使用描述列表或统一详情展示组件。

## i18n

- 所有用户可见文案默认走 i18n。
- 列定义、校验消息、按钮文案、提示文案、标题、描述都必须支持语言切换。
- 如果使用 `computed` 即可满足动态刷新，应优先采用。
